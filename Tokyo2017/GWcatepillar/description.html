GWcatepillar

GWcatepillar:A conceptual animation illustrating how gravitational waves distort space. Created for Nergis' Mavalvala's Kavli lecture.

Parameters

There are several variables to control the appearance and behaviour of this dataset.
  • GWcatRadius: the size of the markers
  • GWcatWaveFactor: the strength of the gravitational wave effect
  • GWcatMarkerAlpha: the opacity of the arkers
  • GWcatMarkerType: sets the type of marker.
    1. Gaussian
    2. soft circle with outline and center dot
    3. soft circle with center dot
    4. ring
    5. hard circle
  • GWcatLength: sets the number of rings in the cylinder. Goes from -25 to 25

The module will appear as "GWcatepillar" in the object tree, under /KavliLecture/Mavalvala.
